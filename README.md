## Installation

- Clone this repository:

  ```sh
  git clone https://github.com/maxxxxxdlp/tetris-react/
  ```

- Install dependencies:

  ```sh
  npm install
  ```

- Run the app:

  ```sh
  # For Development:
  npm run dev
  # For Production:
  npm run build && npm run start
  ```
